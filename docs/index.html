Course Description

The course content found in these webpages was developed by Nathan Birch to fulfill the Performance Based Objectives of the JavaScript Level 1 course developed and managed by the Digital Workshop Center educational institution.

Performance Based Objectives

Upon successful completion of the JavaScript Beginners class, students will be able to:

  • Understand an Introduction to JavaScript

  • Use values, types, variables, and expressions

  • Apply control flow and proper syntax

  • Implement data structures and functions

  • Review the DOM (Document Object Model)

  • Add custom events

  • Build your own form elements and validation

  • Create a simple JavaScript web application

  • Work on an advanced website incorporating a JavaScript project
