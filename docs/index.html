Course Description

The course content found in these webpages was developed by Nathan Birch to fulfill the Performance Based Objectives of the JavaScript Advanced Class (Level 2) course developed and managed by the Digital Workshop Center educational institution.

Instructor Introduction

Performance Based Objectives

Upon successful completion of the JavaScript Beginners class, students will be able to:

  • Expand JavaScript vocabulary

  • Maintain JavaScript code using programming techniques

  • Learn about modules, callbacks, JavaScript evolution

  • Explore dependencies and builds

  • Create dynamic frameworks

  • Review testing and offline storage

  • Understand asynchronous JavaScript

  • Build your portfolio creating advanced JavaScript projects

Course Pre-requisites

Coming into this course

  • Basic digital literacy including file management is required.

  • Web Design with HTML & CSS Beginners is required

  • JavaScript Beginners (Level 1) is required.
