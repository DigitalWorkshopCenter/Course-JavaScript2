Assignment #1

Overview

Today, you will complete your first assignment for this course. You will have an opportunity to demonstrate your understanding of the concepts taught in this module, and surely you will learn a lot as you develop a program of your own. These assignments are designed to be both fun and educational, as well as give you an opportunity to stretch youself and develop your skills.

Your tasks are as follows:

Create the file and run it in the browser:

  • Create a new html file in your text editor

  • Use the ! shortcut shown in the reading material to generate an html template for you to use.

  • Add a script tag to the of the head of your html document. This will look very similar to the hello world activity done previously.

  • Create a js file and add the filename to the script tag listed above.

  • Save the file, and run it in the browser. Ensure that you can make it this far.

Create a friendly JavaScript program

  • Prompt the user for their name. In the class activity, you used the alert function to show a popup in the browser. Now, you will use the prompt() function. The w3schools page on prompt() may be helpful as you do this.

  • Save the user's name in a variable called username

  • Using alert(), send a message to this user that includes both their name and something that you have found to be meaningful in your life.

Stretch Challenges

If you had fun working on this assignment and want to do more, here are some suggestions...

  • Have multiple prompt()s and save multiple variables. Change your message to include all of them.

  • You learned how to do arithmetic in the reading this week, try to add some math into your program. You may want to look into parseFloat().

  • Use an if statement based on user input. Perhaps you prompt them asking if they want two messages or one. Change your output based on the if statement from their choice.

  • Add something to your html file.

Save/push changes to Github

Now that you have finished your program, it is time to save it in Github. We do this so that potential employers can see your work. It is good to be in the habit of saving your code in Github even in the very beginning. Anytime you write a program or work on a project, you will want to do what we are about to do now.

  • Open the terminal in VS code by right clicking the JavaScript1 folder and selecting "Open in Integrated Terminal".

  • execute the following commands:

  • git status - This will show what changes have been made in your repository.

  • git add . - This will prepare all of your changes for a commit (which will be saved in version control if you ever need to view or return to the state that your code is currently in).

  • git commit -m "Lesson 1 assignment" - This will save the changes that you just added with the name that you put for the message, in this case: "Lesson 1 Assignment".

  • git push - This will save your code in Github. Now you could go to the github.com website and view your code their as well along with your commit.

  • Navigate to github.com and verify that your repository shows your latest code.
